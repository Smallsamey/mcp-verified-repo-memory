## 🌟 How mcp-verified-repo-memory Works

The software helps coding agents by keeping track of things they need to remember:

- **Citations:** It shows where information came from for accuracy.
- **JIT Verification:** It checks information right before it’s used.
- **TTL (Time-To-Live):** It removes data that is too old automatically.

These features help keep the data fresh and trustworthy without manual checking.
